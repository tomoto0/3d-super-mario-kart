# Sound Effects - Common

Universal sound effects used across all courses.

## Engine Sounds
- `engine_idle.mp3` - Kart engine idle
- `engine_rev.mp3` - Engine revving

## Item Sounds
- `item_get.mp3` - Item box collection
- `item_use.mp3` - Item activation
- `boost.mp3` - Speed boost
- `boost_big.mp3` - Super boost

## Combat Sounds
- `missile_fire.mp3` - Missile launch
- `missile_hit.mp3` - Missile impact
- `shield_up.mp3` - Shield activation
- `shield_hit.mp3` - Shield impact
- `lightning.mp3` - Lightning strike

## Race Sounds
- `countdown.mp3` - Race countdown (3,2,1)
- `countdown_go.mp3` - Race start (GO!)
- `lap_complete.mp3` - Lap completion
- `race_finish.mp3` - Race finish

## Collision Sounds
- `collision.mp3` - Kart collision
- `crash.mp3` - Major crash
- `spin_out.mp3` - Spin out effect
