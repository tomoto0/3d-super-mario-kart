# Sound Effects - Castle

Castle course-specific sound effects.

## Files
- `ambient.mp3` - Castle dungeon ambience
- `lava.mp3` - Lava bubbling
- `fire.mp3` - Flame effects
- `fireball_throw.mp3` - Fireball projectile
- `fireball_hit.mp3` - Fireball impact
- `thwomp.mp3` - Thwomp crushing sound

## Description
These sounds enhance the dangerous atmosphere of Bowser's Castle and similar courses.
