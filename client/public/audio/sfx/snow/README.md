# Sound Effects - Snow

Snow course-specific sound effects.

## Files
- `ambient.mp3` - Arctic ambience
- `wind.mp3` - Cold wind effects
- `freeze.mp3` - Freezing sound
- `snowball_throw.mp3` - Snowball projectile
- `snowball_hit.mp3` - Snowball impact

## Description
These sounds create the icy atmosphere of Sherbet Land and other snow courses.
