# Music Directory

Background music files for different courses.

## Files

- `grassland.mp3` - Tropical Paradise background music
- `snow.mp3` - Sherbet Land background music
- `castle.mp3` - Bowser's Castle background music
- `rainbow.mp3` - Rainbow Road background music
- `menu.mp3` - Main menu background music

## Format

All music files should be in MP3 format, optimized for web playback:
- Bitrate: 128-192 kbps
- Sample rate: 44.1 kHz
- Duration: 2-4 minutes (looped)
