# Texture Assets

Texture files for 3D rendering in the game.

## Categories

### Environment Textures
- Road surfaces
- Terrain (grass, snow, lava)
- Skyboxes
- Environmental objects

### Kart Textures  
- Kart body materials
- Wheel textures
- Character skins

### UI Textures
- HUD elements
- Menu backgrounds
- Icons and buttons

### Effect Textures
- Particle textures
- Explosion sprites
- Trail textures

## Technical Notes

Most textures are procedurally generated in the code to keep file sizes small.
See `js/textures.js` for the texture generation system.
